# Interface_Sync
Mini project demonstrating an interface sync

Built a synchronisation engine that encodes and decodes Contact, Task, Event, and Note objects using a iSyncable interface.

Demonstrates the use of interface, sync engine and OOP concepts.

Features <br>
Interface Monitoring: Track and log interface activity

Data Synchronisation: Ensure consistency across modules

Error Handling: Detect and report mismatches or failures

Configurable: Supports adding new interfaces easily

Tech Stack <br>
Language: Java
